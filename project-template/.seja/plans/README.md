# Plans

Planos de implementação numerados com steps, checkpoints e verificações.

- **Formato:** `plan-XXXXXX.md` (sequencial)
- **Estrutura D16:** cada step tem `checker: [x]` obrigatório antes do próximo step
- **Status:** `draft` → `approved` → `in_progress` → `done`
- **Roadmaps:** `roadmap-XXXXXX.md` para visão multi-plano

Todo plano aprovado é vinculado a uma feature do `product-design-as-intended.md`.
