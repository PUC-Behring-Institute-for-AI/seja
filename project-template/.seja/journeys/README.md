# Journey Maps (JM)

Mapas de jornada que documentam a interação do usuário com o sistema
sob a ótica da Semiotic Engineering.

## Convenção de Nomenclatura

| Prefixo | Tipo | Quem cria | Descrição |
|---------|------|-----------|-----------|
| **JM-TB** | Top-down Behavior (designed) | @seja-design | Jornada planejada — o que o designer INTENCIONA |
| **JM-E** | Empirical (discovered) | @seja-semiotic-inspector | Jornada descoberta — o que a inspeção semiótica REVELA |

## Formato

Cada step documenta: ator, artefato, comunicabilidade (mensagem I/você).
A diferença entre JM-TB e JM-E revela spec-drift semiótico.
