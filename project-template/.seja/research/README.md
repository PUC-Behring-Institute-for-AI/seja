# Research Reports

Relatórios de pesquisa investigativa que alimentam decisões de design e planos.

- **Formato:** `research-XXXXXX.md`
- **Campos:** question, report_body, recommendation, branch
- **Branch:** `design` (pesquisa para informar design intent) ou `plan` (pesquisa para informar implementação)

A pesquisa ocorre entre as fases de design e plan, quando hipóteses precisam
ser validadas antes do compromisso de implementação.
